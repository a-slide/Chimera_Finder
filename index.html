Chimera_finder

BASH scripts to extract chimeric pairs and chimeric reads from NGS data mixing 2 DNA references

Welcome to Chimera_finder Pages.

Short description Simple bash script written for Debian Unix environments

Download and execute

$ git clone repo_name my_folder/
$ cd my_folder/
$ sudo chmod u+x path1.sh path2.sh ...
$ # For chimeric pairs extraction
$ ./path1.sh
$ # For chimeric splited reads extraction
$ ./path2.sh

Dependencies:

  • Sickle
  • Bowtie2
  • Samtoools
  • Bedtools
