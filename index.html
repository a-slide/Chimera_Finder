Chimera_finder

BASH scripts to extract chimeric pairs and chimeric reads from NGS data mixing 2 DNA references

Welcome to Chimera_finder Page

Motivations

Chimera_finder consist of 2 simple Bash pipelines for illumina NGS data, originally dedicated to retrieval of viral insertion sites retrieval in an host cell genome (chimeric paired end and chimeric reads).

Dependencies:

The script was developed under Linux Mint 16 "petra" but is compatible with other LINUX debian based distribution. The following dependencies are required by the pipeline. Users should verify that all of these programs are added to its PATH and correspond at least to the indicated version.

  • GNU bash, version 4.2.45(1)-release https://www.gnu.org/software/bash/
  • Sickle (v1.210) https://github.com/najoshi/sickle
  • Bowtie2 (v2.1.0) http://bowtie-bio.sourceforge.net/bowtie2/index.shtml
  • Samtools (v0.1.19-96b5f2294a) http://samtools.sourceforge.net/
  • Bedtools (v2.17.0) https://github.com/arq5x/bedtools2/release
